Libraries and Modules

Modules

  • Module ≙ namespace (collection) of method
  • Not a class
  • Can mix method into a class
  • eg: Enumerable
    • Provides most collection operations including each
  • eg: Comparable enables <=> and sort
  • Ensure you read documentation for implementation
  • Defined for a class using include

Gems

  • Gem ≙ ruby library
  • Used with require at the top of a file
  • Bundler solves “library hell”
    • Every project has a Gemfile and a Gemfile.lock
    • Gemfile defines libraries and versions needed for a file
    • Gemfile.lock should be versioned with project

Semantic Versioning (semver)

  • Version A.B.C
    • C ≙ patch / backwards-compatible
    • B ≙ minor backwards-compatible new functionality
    • A ≙ major ‘breaking changes’ in API

Ruby Blocks

  • Iterators define each which yields the value with yield
  • The value yielded is what will be the variable defined in the block

Coding Notes

  • attr_reader only provides a get and not a set
  • Can also use self instead of @
  • send will send a value to a method
